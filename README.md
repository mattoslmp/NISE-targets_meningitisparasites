# NISE-targets_meningitisparasites

Comparative AnEnPi-derived catalog of non-homologous isofunctional enzyme candidates in neurotropic and neurologically relevant helminths.

This repository contains the text-based components of the generated package: scripts, processed CSV files, markdown manuscript and a static HTML summary. The full DOCX/Excel/PNG/SVG/FASTA bundle is available in the downloadable package generated in ChatGPT.

Main results from the generated package:

- 7 parasite datasets were parsed.
- 43 curated EC activities were prioritized.
- 26 target ECs were present as analog candidates in all seven datasets.

Important: the uploaded ZIP did not contain SUPERFAMILY/SCOP fold-assignment tables, so the current release reports AnEnPi-reconstructed candidates and does not claim unobserved fold validation.
