Work summary

This project prioritizes enzyme activities that occur in both human and parasite datasets but are assigned to parasite cluster configurations absent from the human cluster set. The resulting layer is reported as AnEnPi-reconstructed analog candidates and is ranked by recurrence, biological relevance, inhibitor precedent, and curation confidence.

Data-integrity note. The uploaded package did not contain SUPERFAMILY/SCOP fold-assignment tables. Fold validation is therefore implemented as a required confirmation layer, not claimed as completed evidence.

Reproducible workflow

1. Input dataParasite FASTA and AnEnPi output tables
2. EC/cluster parsingReconstruct human and parasite EC-by-cluster matrices
3. Analog callSame EC plus parasite cluster absent from human cluster set
4. Fold validationSUPERFAMILY/SCOP confirmation required
5. Cross-species coreIdentify recurrent ECs across nematode and cestode datasets
6. Drug-target triageRank by recurrence, pathway biology, chemistry and selectivity
7. OutputsArticle, tables, figures, FASTA index, scripts and this web page
