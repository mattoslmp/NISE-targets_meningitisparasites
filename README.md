# NISE-targets_meningitisparasites

Comparative AnEnPi-derived catalog of non-homologous isofunctional enzyme candidates in neurotropic and neurologically relevant helminths.

## GitHub Pages

A static project page is available in `docs/index.html` and is ready for GitHub Pages publication.

Expected public URL after Pages is enabled:

`https://mattoslmp.github.io/NISE-targets_meningitisparasites/`

To activate it in GitHub:

1. Open the repository settings.
2. Go to **Pages**.
3. Select **Deploy from a branch**.
4. Select branch **main** and folder **/docs**.
5. Save and wait for the deployment to finish.

## Repository contents

This repository contains the text-based components of the generated package: scripts, processed CSV files, markdown manuscript notes and a static HTML summary. The full DOCX/Excel/PNG/SVG/FASTA bundle is available in the downloadable package generated in ChatGPT.

## Main results from the generated package

- 7 parasite datasets were parsed.
- 43 curated EC activities were prioritized.
- 26 target ECs were present as analog candidates in all seven datasets.

## Data-integrity statement

The uploaded ZIP did not contain SUPERFAMILY/SCOP fold-assignment tables, so the current release reports AnEnPi-reconstructed candidates and does not claim unobserved fold validation.
