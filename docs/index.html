Project summary

This resource organizes the outputs of an AnEnPi-style workflow to identify enzyme activities shared between humans and parasites where parasite proteins can be assigned to clusters distinct from the human cluster set. These candidates are treated as putative non-homologous isofunctional enzyme layers and are prioritized for downstream fold validation, structural inspection and inhibitor-focused literature review.

The organisms represented in the parsed package are Angiostrongylus cantonensis genome, A. cantonensis transcriptome-derived proteins, Echinococcus granulosus, E. multilocularis, Taenia solium, Toxocara canis and Onchocerca volvulus.

Data integrity note

No unobserved fold-validation result is claimed. The uploaded package did not contain SUPERFAMILY/SCOP fold-assignment output tables. Therefore, this page and the manuscript report AnEnPi-reconstructed candidates and include fold validation as a reproducible required layer for final NISE confirmation.
